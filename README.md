# Проект на Flask представляет собой мессенджер
## На странице регистрации пользователь должен ввести свое имя, пароль, и электронную почту. Это будет храняться в базе данных. На странице My profile располагается общий чат для всех зарегестрированных пользователей, список пользователей и карточка профиля с информацией пользователя. Можно перейти на страницу каждого пользователя кликнув на него. На странице пользователя располагается информация о нем и приватный чат. 
## Локально запускается через файл app.py
